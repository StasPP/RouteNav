Режим перевода точек в формат для приемников Garmin. __Входные данные: точки из произвольного текстового формата; __Выходные данные: файл *.GPX
Быстрое создание прямой линии методом наименьших квадратов (МНК) по траектории движения. __Входные данные: трек в формате *.GPX; __Выходные данные: файл *.TXT с точками начала и конца линии
Быстрое создание прямой линии методом наименьших квадратов (МНК) по траектории движения. __Входные данные: точки трека в произвольном формате *.TXT; __Выходные данные: файл *.TXT с точками начала и конца линии
Создание маршрута и псевдотрека для дальнейшего проецирования на трек. __Входные данные: точки маршрута (WayPoints) в формате *.GPX; __Выходные данные: файл маршрута *.RTS + файл псевдотрека *.GPX
Быстрое проецирование маршрута (с разбиением по проложению) на трек. __Входные данные: трек в формате *.GPX; точки маршрута в произвольном формате *.TXT или *.RTS (если в *.GPX не содержится маршрут) __Выходные данные: трек + точки, спроецированные на него в формате *.GPX
Быстрое проецирование маршрута (с разбиением по проложению) на трек. __Входные данные: точки трека в произвольном формате *.TXT; точки маршрута в произвольном формате *.TXT или *.RTS  __Выходные данные: трек + точки, спроецированные на него в формате *.GPX
Построение высотного профиля для пунктов ГСЗ по усреденной линии. __Входные данные: Точки профиля ГСЗ и пункты взрывов в *.GPX, *.TXT или *.MARK  __Выходные данные: линия осреднения, график высот (в утилите построения графиков) для точек, спроецированных на линию осреднения.
